```
usage: pcdl_make_gif [-h] [path] [interface]

this function generates a gif movie from all image files found in the path
directory in the specified interface file format.

positional arguments:
  path        relative or absolute path to where the images are from which the
              gif will be generated. default is . .
  interface   specify the image format from which the gif will be generated.
              these images have to exist under the given path. they can be
              generated with the plot_scatter or plot_contour function.
              default is jpeg.

options:
  -h, --help  show this help message and exit

homepage: https://github.com/elmbeech/physicelldataloader
```
